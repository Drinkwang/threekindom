# Steam 成就系统开发进度

## 已完成

- 新增全局单例 `AchievementManager`，路径：`res://Scripts/achievement_manager.gd`。
- 已在 `project.godot` 的 `[autoload]` 注册，可在任意脚本中直接调用 `AchievementManager`。
- 已按 `成就准备.bak` 建立 35 个 Steam 成就 ID，对应 `NEW_ACHIEVEMENT_1_0` 到 `NEW_ACHIEVEMENT_2_2`。
- 已给 `saveData` 增加本地成就记录：
  - `steam_achievements`
  - `steam_achievement_unlock_times`
- 兼容旧存档：读档时会自动补齐成就字段。
- 支持 Steam 兼容层：
  - 如果运行环境存在 `Steam` 单例，会调用 `setAchievement` / `set_achievement` / `activateAchievement` 之一解锁，再调用 `storeStats` / `store_stats` 保存到 Steam。
  - 如果没有 Steam 单例，会先写入本地存档并打印日志，编辑器内不会报错。
- 已加入自动检测：
  - 最高难度基建：运粮、水利、城防。
  - 任意派系人数满员。
  - 丹阳、士族、豪族支线完成。
  - 糜家豪宅秘密、黄帝古龙、尸皇陶谦。
  - 仕诡牌成就录全部完成。
  - 打开玄阴秘境、获得霸道之息。
  - 关羽、张飞、无名 10 级。
  - 薪火相传、辕门射戟、抢吕布马。
  - 青梅煮酒、击败曹操、霸道线路完成（`endPath == GameManager.endPath.xuzhou`）。

## 如何调用

在任何 GDScript 里调用：

```gdscript
AchievementManager.unlock("pick_scene_item")
```

如果要一次解锁多个：

```gdscript
AchievementManager.unlock_many(["merchant_gift", "sell_item_to_merchant"])
```

如果想强制重新扫描当前存档状态：

```gdscript
AchievementManager.check_all()
```

如果读档后想把本地已完成成就补同步到 Steam：

```gdscript
AchievementManager.sync_unlocked_to_steam()
```

## 需要手动接入的瞬时触发点

这些成就无法只靠存档状态可靠推断，需要在对应行为发生时调用：

```gdscript
AchievementManager.unlock("enhanced_hundred_schools") # 使用强化后的诸子百家论集
AchievementManager.unlock("enhanced_gift_box")        # 使用强化后的珍品礼盒
AchievementManager.unlock("enhanced_victory_kit")     # 使用强化后的胜战锦囊
AchievementManager.unlock("enhanced_qi_pill")         # 使用强化后的益气丸
AchievementManager.unlock("use_pill_from_inventory")  # 在背包双击使用益气丸/丹药
AchievementManager.unlock("pick_scene_item")          # 主动点击拾取场景物品
AchievementManager.unlock("merchant_gift")            # 序章获得商人赠礼
AchievementManager.unlock("sell_item_to_merchant")    # 向商人卖出物品
AchievementManager.unlock("faction_request")          # 完成一次派系索取
AchievementManager.unlock("tao_heirs_survive")        # 陶氏子弟幸存
AchievementManager.unlock("hard_guanyu_duel")         # 最高难度关羽比武获胜
AchievementManager.unlock("hard_zhangfei_duel")       # 最高难度张飞比武获胜
AchievementManager.unlock("hard_wuming_duel")         # 最高难度无名比武获胜
```
